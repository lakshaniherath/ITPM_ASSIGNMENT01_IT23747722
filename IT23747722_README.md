## Project Description
This project contains an automated test suite developed using **Playwright** and **Python** to evaluate the transliteration accuracy of the Singlish to Sinhala translation tool available at [PixelsSuite Chat Translator](https://www.pixelssuite.com/chat-translator). It processes 50 custom test cases covering all 24 Singlish input types to analyze the accuracy and weaknesses of the system.

## How to Run the Project

1. Open terminal inside project folder

2. (Optional) Activate virtual environment
   venv\Scripts\activate

3. Install dependencies
   pip install -r IT23747722_requirements.txt
   Playwright install

4. Run the automation script
   python IT23747722_test_automation.py --excel "IT23747722Assignmnet1 - Test cases.xlsx" --url "https://www.pixelssuite.com/chat-translator"
